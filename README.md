> JavaScript array inheritance done right.


## Install

```sh
$ npm install --save inherit-array
```


## Usage

```js
var inheritArray = require('inherit-array');

inheritArray('Rainbow');
```

```sh
# creates a browser.js
$ npm run browser
```
